Bienvenido al repositorio de Luxury Repair, plataforma para gestión y promoción de servicios técnicos de reparación de televisores en Madrid.

## Enlaces principales

- **Frontend (Next.js, público):**
  - Producción: [https://luxuryrepair.github.io/luxury/](https://luxuryrepair.github.io/luxury/)
  - Sitemap: [https://luxuryrepair.github.io/luxury/sitemap.xml](https://luxuryrepair.github.io/luxury/sitemap.xml)
  - Robots: [https://luxuryrepair.github.io/luxury/robots.txt](https://luxuryrepair.github.io/luxury/robots.txt)

- **Backend (API Flask):**
  - Producción: [https://luxury-alpha-eight.vercel.app/](https://luxury-alpha-eight.vercel.app/)
  - Documentación y scripts SQL: ver carpeta [`doc/`](doc/)

## Estructura del repositorio

- `frontend/` — Aplicación web Next.js (UI, SEO, rutas públicas y privadas)
- `backend/` — API REST en Flask, lógica de negocio y acceso a datos
- `doc/` — Scripts SQL, documentación técnica y migraciones

## Recursos para rastreadores y validadores

- Sitemap XML y robots.txt generados automáticamente para facilitar la indexación
- URLs canónicas y rutas amigables para SEO
- Documentación técnica enlazada en [`doc/`](doc/)

## Contacto y soporte

Para dudas técnicas, sugerencias o incidencias, consulta la documentación en [`doc/`](doc/) o abre un issue en este repositorio.
